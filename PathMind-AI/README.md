# 🚀 PathMind AI

PathMind AI is an AI-powered Career Intelligence Platform designed to help users analyze resumes, improve ATS scores, simulate interviews, generate career roadmaps, and receive AI-powered career mentorship.

---

## 📄 AI Resume Analyzer
- Upload PDF resumes
- Extract resume content
- Detect technical skills
- Resume intelligence system

## 🎯 ATS Optimization

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/PathMind-AI.git
```

## Navigate to Project

```bash
cd PathMind-AI
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Project

```bash
streamlit run main.py
```
